# 📸 Post Web App | MERN Stack Social Feed Platform

A modern full-stack MERN application that allows users to create, view, and manage posts with image uploads and captions. Built with a clean UI and scalable backend architecture for real-world deployment.

## 🚀 Core Features

✅ Create posts with image upload and caption
✅ View all posts in a dynamic social feed
✅ Delete posts instantly
✅ Responsive UI for desktop & mobile
✅ Cloud image hosting with ImageKit
✅ REST API based backend architecture

## ⚙️ Local Setup Guide

```bash
# Clone Repository
git clone https://github.com/Aditya-Maurya-101/Post-create-web.git
cd Post-create-web

# Run Backend
cd Backend
npm install
npm run server

# Open New Terminal and Run Frontend
cd Frontend
npm install
npm run dev
```
