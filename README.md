# 📸 Post Web App | MERN Stack Social Feed Platform

A modern full-stack MERN application that allows users to create, view, and manage posts with image uploads and captions. Designed with a clean user interface and scalable architecture for real-world deployment.

## 🚀 Core Features

✅ Create posts with image upload and caption  
✅ Browse all posts in a dynamic feed  
✅ Delete posts instantly  
✅ Responsive user interface for desktop & mobile  
✅ Cloud image hosting with ImageKit integration  
✅ REST API based architecture
